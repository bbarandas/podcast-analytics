# Podcast Analytics — Resenha Tática

Toolkit de análise tática, técnica e biomecânica de futebol.

**Páginas:**
- **Posicionamento** — mapas de calor, passes e pressão (StatsBomb Open Data)
- **Vídeo** — extração de frames de lances específicos
- **Biomecânica** — análise de pose e ângulos articulares (MediaPipe)
